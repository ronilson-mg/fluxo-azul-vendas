Perguntas Frequentes

Ainda tem dúvidas?
Respondemos tudo.

O sistema envia mensagens automaticamente? +
Não. O Fluxo Azul prepara a mensagem ideal para cada situação, mas quem decide enviar é sempre você. Isso garante controle total e blindagem jurídica completa. Você clica no botão, o WhatsApp abre e você confirma o envio.
É seguro? Meus dados e dos meus clientes ficam protegidos? +
Sim. Utilizamos Supabase com criptografia de nível bancário, conformidade total com a LGPD e isolamento de dados por empresa. Nenhum dado é compartilhado com terceiros.
Preciso saber de tecnologia para usar? +
De forma alguma. O Fluxo Azul foi desenhado para quem não tem nenhum conhecimento técnico. Se você sabe usar WhatsApp, você sabe usar o Fluxo Azul. A interface é intuitiva e em português.
Tem garantia? E se eu não gostar? +
Você tem 14 dias completamente grátis para testar tudo sem precisar de cartão de crédito. Se não gostar, simplesmente não assina. Sem letras miúdas, sem comprometimento.
Funciona no celular? +
Sim! O Fluxo Azul tem interface otimizada para celular com menu inferior estilo app, botão "Cobrar Agora" sempre visível e cards intuitivos. Funciona perfeitamente no iPhone e Android.
Como funciona o pagamento após o trial? +
Após os 14 dias, você escolhe o plano ideal e paga via Pix mensalmente. O processo é simples e direto pelo WhatsApp. Não há cobrança automática — você renova quando quiser.
